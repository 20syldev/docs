# Chat en ligne

L'endpoint `/chat` fournit un système de messagerie en temps réel avec support pour les canaux publics et privés.

## Endpoints associés

- [/v4/chat](./chat/messages) - Récupérer les messages récents
- [/v4/chat](./chat/send) - Envoyer un message
- [/v4/chat/private](./chat/private) - Créer et gérer des canaux de chat privés
- <Method type="delete" sm /> [/v4/chat/:token](./chat/clear) - Effacer les messages d'un chat privé
