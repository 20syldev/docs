# 4.6.0

## API

Aucun changement API dans cette version.

## Documentation

- Ajout du paramètre requis `session` dans le panel **[POST /chat](/v4/fr/chat)**.
- Ajout de l'option `true` au paramètre `avatar` dans le panel **[GET /placeholder](/v4/fr/placeholder)**.
- Ajout des options de fuseau horaire en dropdown dans le panel **[GET /time](/v4/fr/time)**.
- Le bouton **Envoyer** des panels Try est désormais désactivé tant que tous les paramètres requis ne sont pas remplis.
- Les panels Try résolvent désormais leur configuration depuis un registre d'endpoints centralisé plutôt que via des props inline.
- Correction du footer recouvert par le sommaire **Sur cette page**.
- Ajout d'arrondissements aux coins des tableaux de documentation.
