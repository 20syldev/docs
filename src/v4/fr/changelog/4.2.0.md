# 4.2.0

## API

**[API v4.2.0](https://github.com/20syldev/api/releases/tag/4.2.0)** — Encode, Text, Validate.

- Ajout de l'endpoint **[GET /encode](/v4/fr/encode)** pour encoder et décoder du texte (Base64, URL, Morse, ROT-13, César, binaire).
- Ajout de l'endpoint **[GET /text](/v4/fr/text)** avec 4 méthodes : slug, statistiques texte, Lorem Ipsum, nombre en lettres (FR/EN).
- Ajout de l'endpoint **[GET /validate](/v4/fr/validate)** pour valider des formats de données (Luhn, IBAN, email).
- Ajout de tests unitaires pour encode, text et validate.
- Ajout de tests d'intégration HTTP pour les 3 nouveaux endpoints.
