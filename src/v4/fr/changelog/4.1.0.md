# 4.1.0

## API

**[API v4.1.0](https://github.com/20syldev/api/releases/tag/4.1.0)** — Dice, Statistics, Roman.

- Ajout de l'endpoint **[GET /dice](/v4/fr/dice)** pour lancer des dés en notation RPG (`2d6+3`).
- Ajout de l'endpoint **[GET /statistics](/v4/fr/statistics)** pour les statistiques descriptives (moyenne, médiane, écart-type...).
- Ajout de la méthode **roman** à l'endpoint **[/algorithms](/v4/fr/algorithms)** pour la conversion bidirectionnelle chiffres romains / nombres.
- Lecture dynamique de la version depuis `package.json` via `APP_VERSION`.
- Ajout d'une suite de tests complète : 19 fichiers unitaires + tests d'intégration HTTP.

## Documentation

- Ajout des pages de documentation pour [Dice](/v4/fr/dice) et [Statistics](/v4/fr/statistics).
- Ajout de la méthode `roman` dans la documentation d'[Algorithms](/v4/fr/algorithms).
- Ajout du composant **Footer** avec liens changelog, GitHub et issues.
- Consolidation des couleurs de badges et cards avec des **variables CSS**.
- Amélioration du layout des composants **Panel** et **Playground** (flex ordering, wrap, gap).
- Réorganisation des **media queries** responsive (Mobile / Tablet / Desktop / Wide).
- Masquage du **Terminal** sur mobile (< 768px).
- Correction de l'affichage des **tableaux** et du **header API** (flex-wrap).
- Réorganisation de l'ordre des **imports CSS** pour une meilleure cascade.
