# Online Chat

The `/chat` endpoint provides a real-time messaging system with support for public and private channels.

## Related Endpoints

- [/v4/chat](./chat/messages) - Retrieve all recent messages
- [/v4/chat](./chat/send) - Send a message
- [/v4/chat/private](./chat/private) - Create and manage private chat channels
- <Method type="delete" sm /> [/v4/chat/:token](./chat/clear) - Clear all messages from a private chat
