# 4.4.0

## API

**[API v4.4.0](https://github.com/20syldev/api/releases/tag/4.4.0)** — QR Code, Captcha, Color, Convert, Hash, Placeholder.

- Improved **[GET /qrcode](/v4/en/qrcode)**: PNG output by default, new options (`size`, `margin`, `correction`, `dark`, `light`, `icon`, `iconSize`, `iconPadding`, `iconRadius`, `format`), icon overlay support with HTTPS validation and 2MB size limit.
- Improved **[GET /captcha](/v4/en/captcha)**: new options (`length`, `width`, `height`, `noise`, `bg`, `color`), auto-generates random text when none is provided, `X-Captcha-Text` response header for server-side verification.
- Improved **[GET /color](/v4/en/color)**: accepts optional `?hex=` input to convert a specific color instead of generating a random one.
- Expanded **[GET /convert](/v4/en/convert)**: new unit categories — length (km, mi, m, ft, cm, in, yd), weight (kg, lb, oz, g, ton), data (b, kb, mb, gb, tb), speed (km/h, mph, m/s, knots). Absolute zero validation now only applies to temperature units.
- Improved **[POST /hash](/v4/en/hash)**: new `encoding` option (`hex` or `base64`), throws on unsupported method instead of returning error object, text validation on empty input.
- Improved **[GET /placeholder](/v4/en/placeholder)**: skeleton `lineH` and `gap` now scale proportionally to available height.
- Extracted shared color and helper utilities into dedicated modules (`colors.ts`, `helpers.ts`).
- Added `since(version, min)` helper for version branching in routes.
- Added JSDoc to all utility functions.

## Documentation

- Updated documentation for [QR Code](/v4/en/qrcode), [Captcha](/v4/en/captcha), [Color](/v4/en/color), [Convert](/v4/en/convert) and [Hash](/v4/en/hash).
- Added **[POST /chat](/v4/en/chat)** main page with full documentation, merged redundant send sub-page.
- Renamed Convert from "Temperature Conversion" to "Unit Conversion".
- Added new parameters to **Playground** registry for all updated endpoints.
- Added **Method** badges to all related endpoint links across Chat and Tic-Tac-Toe pages.
- **Try** panel now displays image responses (PNG, SVG) instead of raw binary data.
- Sidebar automatically scrolls to the active link on navigation.
- Centralized language constants in **i18n** module, removed hardcoded language checks.
