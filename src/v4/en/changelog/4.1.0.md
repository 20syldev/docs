# 4.1.0

## API

**[API v4.1.0](https://github.com/20syldev/api/releases/tag/4.1.0)** — Dice, Statistics, Roman.

- Added endpoint **[GET /dice](/v4/en/dice)** to roll dice using RPG notation (`2d6+3`).
- Added endpoint **[GET /statistics](/v4/en/statistics)** for descriptive statistics (mean, median, standard deviation...).
- Added **roman** method to **[/algorithms](/v4/en/algorithms)** for bidirectional Roman numeral / number conversion.
- Dynamic version reading from `package.json` via `APP_VERSION`.
- Added full test suite: 19 unit test files + HTTP integration tests on all endpoints.

## Documentation

- Added documentation pages for [Dice](/v4/en/dice) and [Statistics](/v4/en/statistics).
- Added `roman` method to [Algorithms](/v4/en/algorithms) documentation.
- Added custom **Footer** component with changelog, GitHub and issue links.
- Consolidated badge and card colors with **CSS variables**.
- Improved **Panel** and **Playground** layout (flex ordering, wrap, gap).
- Reorganized **responsive media queries** (Mobile / Tablet / Desktop / Wide).
- Hidden **Terminal** on mobile (< 768px).
- Fixed **table display** and **API header** wrapping.
- Reordered **CSS imports** for better cascade.
