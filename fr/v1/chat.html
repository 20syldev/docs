API REST : Chat en ligne

GET/v1/chat

L'endpoint /chat récupère tous les messages envoyés durant la dernière heure. Cet endpoint ne nécessite aucune option.

Requête
GET /v1/chat 
[
  {
    "username": "User1",
    "message": "Salut !",
    "timestamp": "2025-01-21T08:27:11.068Z"
  }
  {
    "username": "User3",
    "message": "Ça va ?",
    "timestamp": "2025-01-21T08:28:43.070Z"
  }
  {
    "username": "User8",
    "message": "Ouais super et vous ?",
    "timestamp": "2025-01-21T08:32:07.215Z"
  }
]
cURL
$ curl -X GET 'https://api.sylvain.pro/v1/chat'
POST/v1/chat

L'endpoint /chat permet l'envoi d'un message vers le chat global ou un chat privé.

Paramètres
  • username (requis) : Le nom d'utilisateur qui a envoyé le message.
  • message (requis) : Le message à envoyer.
  • session (requis) : L'identifiant de session de l'utilisateur.
  • token (optionnel) : La clé permettant la connexion à un chat privé. Si elle n'est pas fournie, le message sera envoyé dans le chat global.
Requête
$ curl -X POST -d 'username=User3&message=Ça va ?&session=x82Azo-93sZ0d&token=41a43360-9874-4c7a-9ca8-eec29e765a0e' 'https://api.sylvain.pro/v1/chat' 
{
  "message": "Message sent successfully"
}
Si la valeur de username, message ou session est vide, une erreur sera renvoyée.
