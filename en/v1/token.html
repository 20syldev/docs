REST API: Token Generation

POST/v1/token

The /token endpoint generates secure strings called tokens. This endpoint is particularly useful for creating random identifiers for various applications.

Parameters
  • len (optional): The length of the token to generate. Must be an integer between 12 and 4096.
  • type (optional): The type of token to generate. The default type is alphanum.
  • Available options:
    • alpha: Letters only (lowercase and uppercase).
    • alphanum: Letters and numbers.
    • base64: Base64 encoding.
    • hex: Hexadecimal digits.
    • num: Numbers only.
    • punct: Punctuation characters.
    • urlsafe: URL-safe characters.
    • uuid: Universally Unique Identifier (UUID).
Request
POST /v1/token?len=24&type=alphanum 
{
  "token": "FfkfUDfVzRAiNnoMxLtNschm"
}
cURL
$ curl -d 'len=24&type=alphanum' https://api.sylvain.pro/v1/token
If the len value is out of bounds or the type is invalid, a 12-character alphanumeric token will be generated.
