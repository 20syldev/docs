REST API : Online chat

GET/v1/chat

The /chat endpoint retrieves all messages sent during the last hour. This endpoint does not require any options.

Request
GET /v1/chat 
[
  {
    "username": "User1",
    "message": "Hi!",
    "timestamp": "2025-01-21T08:27:11.068Z"
  },
  {
    "username": "User3",
    "message": "How are you?",
    "timestamp": "2025-01-21T08:28:43.070Z"
  },
  {
    "username": "User8",
    "message": "Yeah great, and you?",
    "timestamp": "2025-01-21T08:32:07.215Z"
  }
]
cURL
$ curl -X GET 'https://api.sylvain.pro/v1/chat'
POST/v1/chat

The /chat endpoint allows sending a message to the global chat or a private chat.

Parameters
  • username (required) : The username of the user who sends the message.
  • message (required) : The message to send.
  • session (required) : The session ID of the user.
  • token (optional) : The key used to connect to a private chat. If not provided, the message will be sent to the global chat.
Request
$ curl -X POST -d 'username=User3&message=How are you?&session=x82Azo-93sZ0d&token=41a43360-9874-4c7a-9ca8-eec29e765a0e' 'https://api.sylvain.pro/v1/chat' 
{
  "message": "Message sent successfully"
}
If the value of username, message or session is empty, an error will be returned.
