REST API: Hash

POST /v3/hash

The /hash endpoint convert a string into a hash, i.e. a string of characters that cannot be reversed into the original string.

Parameters
  • text (required) : The string to hash, can be a word, a sentence, a paragraph...
  • method (required) : The hashing method to use.
    • RSA-MD5
    • RSA-RIPEMD160
    • RSA-SHA1
    • RSA-SHA1-2

Request
$ curl -X POST -d "text=hello&method=sha512" "https://api.sylvain.pro/v3/hash" 
{
  "method": "sha512",
  "hash": "9b71d224bd62f3785d96d46ad3ea3d73319bfbc2890caadae2dff72519673ca72323c3d99ba5c11d7c7acc6e14b8c5da0c4663475c2e5c3adef46f73bcdec043"
}
If the value of text or method is empty or invalid, an error will be returned.
