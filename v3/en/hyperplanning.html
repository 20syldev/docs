REST API: Hyperplanning

POST /v3/hyperplanning

The /hyperplanning endpoint allows you to display a Hyperplanning schedule from an ICS file (calendar). Useful for students who want to display their timetable directly on their website.

Parameters
  • url (required): The URL of the ICS file to use and display.
  • detail (optional): Allows you to specify the level of detail for events.
    • full: Displays a complete summary with subject, teacher, classes, date, etc.
    • list: Displays all data in a list.
Request
$ curl -X POST -d "url=https%3A%2F%2Fexample.com%2Ffile.ics&detail=full" "https://api.sylvain.pro/v3/hyperplanning" 
[
  {
  "summary": [
    "Mathematics",
    "DOE",
    "Class",
    "1",
    "Class",
    "2"
  ],
  "subject": "Mathematics",
  "teacher": "DOE",
  "classes": [
    "Class 1",
    "Class 2"
  ],
  "start": "2000-03-01T12:00:00.000",
  "end": "2000-03-01T14:00:00.000"
  }
]
If the value of url is empty, an error will be returned.
