REST API: Tic-Tac-Toe List

POST /v3/tic-tac-toe/list

The /tic-tac-toe/list endpoint allows you to retrieve all public Tic-Tac-Toe games currently in progress. This endpoint does not return private games or finished games.

Parameters

This endpoint does not require any parameters. It returns all public games that are currently active.

Request
$ curl -X POST "https://api.sylvain.pro/v3/tic-tac-toe/list" 
{
  "message": "Public games available",
  "count": 2,
  "games": [
    {
      "id": "ABCDE",
      "players": ["User1", "User2"],
      "playersCount": 2,
      "moves": 3,
      "turn": "User1",
      "status": "ready",
      "creation": 1696204800000
    },
    {
      "id": "FGHIJ",
      "players": ["User3"],
      "playersCount": 1,
      "moves": 1,
      "turn": "User3",
      "status": "waiting",
      "creation": 1696204700000
    }
  ]
}
Response Fields
  • message: Confirmation message
  • count: Number of public games available
  • games: Array of game objects with the following properties:
    • id: Unique game identifier
    • players: Array of player usernames in the game
    • playersCount: Number of players currently in the game (0-2)
    • moves: Number of moves made in the game
    • turn: Username of the player whose turn it is
    • status: Current game status (waiting or ready)
    • creation: Timestamp when the game was created
Game Status Values
  • waiting: Game has less than 2 players and is waiting for more players to join
  • ready: Game has 2 players and is ready to play
Empty List Response
{
  "message": "Public games available",
  "count": 0,
  "games": []
}
The list only includes public games that are currently in progress (not finished). Games are sorted by creation time with the newest games first. Private games are never included in this list for privacy reasons. You can join any listed game by using its ID with the /tic-tac-toe endpoint.
