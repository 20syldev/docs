API REST : Hyperplanning

POST /v3/hyperplanning

L'endpoint /hyperplanning permet d'afficher un planning Hyperplanning à partir d'un fichier ICS (calendrier). Utile pour les étudiants qui souhaitent afficher leur emploi du temps directement sur leur site web.

Paramètres
  • url (requis) : L'URL du fichier ICS à utiliser puis à afficher.
  • detail (optionnel) : Permet de spécifier le niveau de détail des événements.
    • full : Affiche un résumé complet avec matière, enseignant, classes, date, etc.
    • list : Affiche toutes les données dans un tableau.
Requête
$ curl -X POST -d "url=https%3A%2F%2Fexemple.com%2Ffchier.ics&detail=full" "https://api.sylvain.pro/v3/hyperplanning" 
[
  {
    "summary": [
      "Mathématiques",
      "DOE",
      "Classe",
      "1",
      "Classe",
      "2"
    ],
    "subject": "Mathématiques",
    "teacher": "DOE",
    "classes": [
      "Classe 1",
      "Classe 2"
    ],
    "start": "2000-03-01T12:00:00.000",
    "end": "2000-03-01T14:00:00.000"
  }
]
Si la valeur de url est vide, une erreur sera renvoyée.
