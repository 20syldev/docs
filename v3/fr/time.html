API REST : Informations sur le temps

GET /v3/time

L'endpoint /time permet de récupérer des informations sur le temps en fonction de différents paramètres. Vous pouvez obtenir l'heure actuelle ou une heure aléatoire entre une plage spécifiée, dans différents formats et fuseaux horaires.

Paramètre
  • type (optionnel) : Le type de temps à récupérer.
    • live (par défaut) : Obtenir des informations sur le temps actuel.
    • random : Obtenir des informations aléatoires sur le temps.
  • start (optionnel) : La date de début de la plage pour une date aléatoire, sous le format YYYY-MM-DD.
  • end (optionnel) : La date de fin de la plage pour une date aléatoire, sous le format YYYY-MM-DD.
  • format (optionnel) : Le format de la réponse.
    • iso (par défaut) : Afficher la date au format ISO 8601.
    • unix : Afficher la date sous forme de timestamp Unix.
    • locale : Afficher la date au format local.
    • date : Afficher la date au format MM/DD/YYYY.
    • time : Afficher l'heure au format HH:MM:SS.
    • year : Afficher l'année.
    • month : Afficher le mois.
    • day : Afficher le jour.
    • hour : Afficher l'heure.
    • minute : Afficher les minutes.
    • second : Afficher les secondes.
    • ms : Afficher les millisecondes.
    • dayOfWeek : Afficher le jour de la semaine.
    • dayOfYear : Afficher le jour de l'année.
    • weekNumber : Afficher le numéro de la semaine.
    • timezone : Afficher le fuseau horaire.
    • timezoneOffset : Afficher le décalage horaire.
  • timezone (optionnel) : Le fuseau horaire à utiliser pour la réponse.
    • UTC (par défaut) : Temps universel coordonné.
    • Europe/Paris: Fuseau horaire de Paris.
    • America/New_York: Fuseau horaire de New York.
    • Asia/Tokyo: Fuseau horaire de Tokyo.
    • Australia/Sydney: Fuseau horaire de Sydney.
Requête
GET /v3/time 
{
  "iso": "2000-01-01T12:00:00.000Z",
  "utc": "Sat, 01 Jan 2000 12:00:00 GMT",
  "timestamp": 946724400,

  "date": "3/3/2000",
  "time": "12:00:00 PM",
  "year": 2000,
  "month": 1,
  "day": 1,
  "hour": 12,
  "minute": 00,
  "second": 0,
  "ms": 0,
  "dayOfWeek": 6,
  "dayOfYear": 1,
  "weekNumber": 1,
  "timezone": "UTC",
  "timezoneOffset": 0

cURL
$ curl -X GET "https://api.sylvain.pro/v3/time?type=random&format=locale&timezone=Europe/Paris" 
{
  "date": "7/28/2024, 2:06:52 PM GMT+01:00"
}
Si la valeur de type, start, end, format ou timezone invalide lorsque vous les spécifiez, une erreur sera renvoyée.
