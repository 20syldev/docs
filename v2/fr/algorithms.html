API REST : Algorithmes

GET /v2/algorithms

L'endpoint /algorithms propose plusieurs algorithmes à entrer en paramètre. Il est possible d'entrer une ou deux valeurs en fonction de l'algorithme.

Paramètres
  • method (requis) : L'algorithme à utiliser.
    • anagram : Vérifie si deux chaînes sont des anagrammes. Nécessite value2.
    • bubblesort : Trie un tableau de nombres, séparés par des virgules.
    • factorial : Calcule la factorielle d'un nombre situé entre 0 et 170.
    • gcd : Calcule le plus grand diviseur commun de deux nombres. Nécessite value2.
    • isprime : Vérifie si un nombre est premier.
    • fibonacci : Calcule une séquence de Fibonacci jusqu'à un nombre donné.
    • palindrome : Vérifie si une chaîne est un palindrome.
    • primelist : Retourne une liste de nombres premiers jusqu'à un nombre donné.
    • primefactors : Retourne la liste des facteurs premiers d'un nombre.
    • reverse : Inverse une chaîne de caractères.
  • value (requis) : La valeur à utiliser dans l'algorithme. Doit contenir au minimum un caractère.
  • value2 (optionnel) : La seconde valeur à utiliser si nécessaire. Doit contenir au minimum un caractère.
Requête
GET /v2/algorithms?method=gcd&value=17&value2=493 
{
  "answer": 17
}
cURL
$ curl -X GET "https://api.sylvain.pro/v2/algorithms?method=gcd&value=17&value2=493"
Si la valeur de method ou value est vide ou invalide, ou que value2 est nécessaire mais vide, une erreur sera renvoyée.
