API REST : Conversion

GET/v2/convert

L'endpoint /convert permet de convertir plusieurs unités de température entre elles. Il faut entrer une valeur à convertir, une unité de départ, et l'unité dans laquelle vous souhaitez la convertir.

Paramètres
  • value (requis) : La valeur à convertir.
  • from (requis) : L'unité de départ.
    • celsius : Unité de départ en degrés Celsius.
    • fahrenheit : Unité de départ en degrés Fahrenheit.
    • kelvin : Unité de départ en Kelvin.
  • to (requis) : L'unité dans laquelle vous souhaitez convertir.
    • celsius : Unité de conversion en degrés Celsius.
    • fahrenheit : Unité de conversion en degrés Fahrenheit.
    • kelvin : Unité de conversion en Kelvin.
Requête
GET /v2/convert?value=30&from=celsius&to=kelvin 
{
  "from": "celsius",
  "to": "kelvin",
  "value": 30,
  "result": 303.15
}
cURL
$ curl -X GET 'https://api.sylvain.pro/v2/convert?value=30&from=celsius&to=kelvin'
Si la valeur de value, from ou to est vide ou invalide, une erreur sera renvoyée.
