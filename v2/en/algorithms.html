REST API: Algorithms

GET/v2/algorithms

The /algorithms endpoint provides several algorithms to use as parameters. You can input one or two values depending on the algorithm.

Parameters
  • method (required): The algorithm to use.
    • anagram: Checks if two strings are anagrams. Requires value2.
    • bubblesort: Sorts an array of numbers, separated by commas.
    • factorial: Calculates the factorial of a number between 0 and 170.
    • gcd: Computes the greatest common divisor of two numbers. Requires value2.
    • isprime: Checks if a number is prime.
    • fibonacci: Calculates a Fibonacci sequence up to a given number.
    • palindrome: Checks if a string is a palindrome.
    • primelist: Returns a list of prime numbers up to a given number.
    • primefactors: Returns the list of prime factors of a number.
    • reverse: Reverses a string.
  • value (required): The value to use in the algorithm. Must contain at least one character.
  • value2 (optional): The second value to use if necessary. Must contain at least one character.
Request
GET /v2/algorithms?method=gcd&value=17&value2=493 
{
  "answer": 17
}
cURL
$ curl -X GET 'https://api.sylvain.pro/v2/algorithms?method=gcd&value=17&value2=493'
If the value of method or value is empty or invalid, or if value2 is required but empty, an error will be returned.
